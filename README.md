🇹🇹 Gastronomia de Trinidad y Tobago - Smart Contract (Base Mainnet)
Este es el vigesimocuarto contrato inteligente de mi coleccion profesional "Sabores de America". Trinidad y Tobago ofrece una de las gastronomias mas diversas del Caribe, fusionando raices africanas e indias con especias locales. Este contrato introduce el campo de Tipo de Chutney, capturando el alma de sus salsas frutales que equilibran perfectamente el calor del curry y el picante del chile habanero.

Hito de la Serie: Vigesimocuarto bloque completado. Se utiliza la variable tipoChutney para clasificar el acompañamiento frutal (ej. Mango, Tamarindo, Pepino), un elemento critico en la armonia de sabores de la isla.

Integridad del Codigo: Arquitectura lineal diseñada para evitar errores de desbordamiento y optimizar el uso de gas en la red Base.

Cero Caracteres Especiales: Codigo limpio de enies y acentos para garantizar una lectura tecnica impecable en cualquier explorador de bloques.

Licencia Abierta: Implementado bajo SPDX-MIT para fomentar la transparencia en el ecosistema Web3.

📖 Modo de Uso

Consultar los Doubles: En la pestaña Read Contract, usa consultarPlato (ID 1) para verificar como el Chutney de Mango y Tamarindo es el protagonista del primer registro.

Registrar Tradicion: En la pestaña Write Contract, usa registrarPlato para añadir iconos como "Bake and Shark" (Chutney: Cilantro) o "Pholourie" (Chutney: Tamarindo).

Votacion Inmutable: Sistema de Likes y Dislikes para medir la popularidad de la gastronomia de Trinidad y Tobago en la blockchain.
